# ⚡ FlyerGenius

Brief flyer IA pour commerçants francophones — FR 🇫🇷 MA 🇲🇦 DZ 🇩🇿 TN 🇹🇳 BE 🇧🇪 CA 🇨🇦

## Live
👉 https://diabizimpression-dev.github.io/flyergenius

## Features
- 🤖 Brief complet généré par Claude (Anthropic)
- 🌙 Modes saisonniers : Ramadan, Aïd el-Fitr, Aïd el-Adha, Été, Rentrée, Fêtes
- 🖨️ 5 formats : A5 Print, Story Instagram, Post Facebook, WhatsApp, Menu A4
- 🎨 Prompt Midjourney/Firefly inclus dans chaque brief
- 📲 Partage direct WhatsApp
- 10 types de commerce, 6 pays, palette hex complète

## Setup
1. Obtenir une clé API Anthropic : https://console.anthropic.com/settings/keys
2. Ouvrir `index.html` → entrer la clé au premier lancement
3. La clé est stockée en sessionStorage (non persistée)

## Stack
- Vanilla HTML/CSS/JS — zéro dépendance
- Claude Sonnet (Anthropic API)
- GitHub Pages ready

## Deploy GitHub Pages
Settings → Pages → Branch: main → / (root) → Save
